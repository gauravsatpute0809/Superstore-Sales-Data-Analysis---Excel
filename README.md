# 📊 Sales Dashboard – Excel

## 🧾 Overview

This project is a Sales Dashboard created in Excel to analyze business performance using sales data. The dashboard provides insights into product performance, profit trends, customer behavior, and regional sales distribution.

---

## 🛠️ What I Worked On

* Imported and cleaned sales data
* Performed data transformation using Excel tools
* Created pivot tables and data structure
* Used formulas for calculations
* Designed an interactive dashboard with filters and slicers

---

## 📊 Dashboard Features

* 📌 Sales by Sub-Category
* 📈 Profit trends over time (by category)
* 🌍 Sales by state (map/chart view)
* 👥 Customer count over years
* 🥇 Top 5 customers by profit
* 📅 Monthly sales trend

---

## 📈 Key Insights

* Phones and Chairs are top-selling products
* Technology category generates highest profit
* Sales are strong in specific states like California
* Customer base is growing year by year
* Few customers contribute major profit
* Sales peak during last months of the year
